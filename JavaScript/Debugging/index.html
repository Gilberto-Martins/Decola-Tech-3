Erros relacionados ao Documento Object Model (DOM).

Throw

O Throw é usado quando desejamos tratar um erro que é esperado.
Ele Retorna o erro em formato de erro pelo console passando as informações que foram espeçificadas em seu tratamento.

Try...catch

O try/catch consistem em dois blocos, o primeiro(try) ira rodar o script, se ouver erro ele ira passar para o segundo bloco(catch), neste devemos encontra a mensagem de erro que ira reporta como console ou usando o throw(preferencia).

Finally

É uma função que sera chamada se tiver ou não um erro, ele é usando junto com o try/catch/finally logo no final.
Mesmo tendo erro ou não ele sera executado.
